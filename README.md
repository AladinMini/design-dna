# design-dna

Reverse-engineered design patterns from the best websites in the world.

Not inspiration boards. Not screenshots. **The actual DNA** — why great designs work and how they're built.

## What's in here

Each entry documents:
- **Why it works** — the psychology, the feeling, the problem it solves
- **How it's built** — stripped-down code, the core technique
- **When to use it** — context for applying the pattern

## Structure

```
/sites          # Studies by company
  /stripe
  /apple
  /airbnb
  /linear
  ...

/patterns       # Cross-site patterns
  scroll-effects.md
  color-transitions.md
  micro-interactions.md
  ...
```

## Philosophy

> "Copy 100 Stripe screens to learn design" — Mengto

But copying isn't enough. Understanding is.

We reverse-engineer. We strip out the essence. We document the why alongside the how.
